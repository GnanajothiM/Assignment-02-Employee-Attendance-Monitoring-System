# Assignment-02-Employee-Attendance-Monitoring-System

A simple Python project to monitor employee attendance and generate a summary report.

Features
Classifies attendance as:
Excellent (90 and above)
Good (75–89)
Improvement Required (Below 75)
Uses break to stop data entry when -1 is entered.
Uses continue to skip records when 0 is entered.
Uses pass as a placeholder for future validation.
Displays attendance summary.
Generates additional insights:
Highest Attendance Percentage
Lowest Attendance Percentage
Average Attendance Percentage
Percentage of Employees in Each Category
